# archive-tool

A simple bash script to create backups for given files to a specific location.


## Usage

```bash
git clone https://github.com/klkkristof/System_Tasks.git && cd Archive_Tool
```

```bash
chmod +x archivetool.sh
```

```bash
./archivetool.sh /path/to/directory
```

## More

You can schedule automatic backups with crontab or add an alias for easier access. :D
